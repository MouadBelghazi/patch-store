# Patch Silicone Shopify Theme Starter

A lightweight, Shopify-friendly single-product storefront for reusable silicone wrinkle patches.

## Included

- **Homepage template** focused on one hero product (`templates/index.json`)
- **Editable content section** in Theme Editor (`sections/one-product-landing.liquid`)
- **Simple responsive styling** (`assets/theme.css`)
- **Minimal theme shell** (`layout/theme.liquid`)

## How to upload quickly

1. Zip this folder contents.
2. In Shopify Admin: **Online Store → Themes → Add theme → Upload zip file**.
3. Open **Customize** and set the **Featured product** in the “One Product Landing” section.
4. Update text blocks if needed and publish.

## Notes

- This is designed for speed and clarity for a one-product launch.
- Keep product claims compliant with your market's advertising rules.
